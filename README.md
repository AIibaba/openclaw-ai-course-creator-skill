*Auto-generate complete online courses with outlines, content, exercises, and marketing plans.*

A full-stack course creation assistant that takes you from topic idea to published course. Handles curriculum design, lesson scripting, quiz generation, and even launch marketing. Built for educators, coaches, and knowledge entrepreneurs on platforms like Udemy, Teachable, and Skillshare.

## Features

| Feature | Description |
|---------|-------------|
| Curriculum Architect | Designs structured course outlines with modules, lessons, and learning objectives |
| Lesson Content Generator | Creates detailed lesson scripts, slide content, and instructor notes |
| Quiz & Exercise Builder | Auto-generates quizzes, assignments, and hands-on exercises aligned with learning goals |
| Student Journey Designer | Maps the complete learning path with prerequisites, milestones, and completion criteria |
| Launch Marketing Kit | Produces course descriptions, promotional emails, and social media campaigns |
| Platform Optimizer | Tailors content and metadata for Udemy, Teachable, Skillshare, and other platforms |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-course-creator-skill
```

## Usage

```
design course [topic]      # Generate complete course outline
write lesson [module]      # Create lesson content and scripts
generate quiz [lesson]     # Build quizzes and exercises
launch plan [course]       # Create marketing and launch strategy
optimize [platform]        # Optimize for specific course platform
```
